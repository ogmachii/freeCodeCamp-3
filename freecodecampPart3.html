HTML Basics and Needs

The Basics:

Below is a quick bullet point of the general idea of what HTML is and how it affects the end user experience with technology:

  • HTML stands for Hyper Text Markup Language.
  • HTML is the standard markup language for creating Web pages, like this one!
  • HTML is the framework of a webpage, the backbone.
  • HTML is essentially a bunch of elements put together.
  • HTML elements tell the browser how to display content.
  • HTML elements describe what content the end user is seeing.
  • HTML elements are defined by a start tag, some content, and an end tag.

The Needs:

The things you'll need to write HTML:

  1. A computer
  2. A text editor i.e. Notepad (Windows), Notepad++, TextEdit (Mac)
  3. Web Browser i.e. Google Chrome, Microsoft Edge (Windows), Safari (Mac)

      4. A computer is a crucial part to the whole process, storing the file, viewing the file, and writing it. A text editor allows one to easily write the file and save it, while the Web Browser allows one to view the file. With those three things anyone can create a web page!

A Bare Bones HTML Webpage

Every HTML webpage will start a little something like this:

<!DOCTYPE html> <html>  <head>  </head>  <body>  </body> </html>

To start with the first line, all HTML documents will start with "<!DOCTYPE html>" this is considered a document declaration.

The first element of the HTML document is our next line known as the HTML element. With the second line and last line of the page respectively, "<html>","</html>", our webpage is defined.

HTML metadata (Data that provides information about other data.* ) is defined within "<head>", "</head>" known as the Head Element. This can be useful to tell the web browser the Character Encoding, Document title, a styling, and much more!

Lastly we have the Body element. Defined with the fifth and sixth lines, "<body>", "</body>" everything within this element will be showed to the user.

Adding Text

To add text to a webpage we will start with our bare bones layout:

<!DOCTYPE html> <html>  <head>  </head>  <body>  </body> </html>

Within the body element, the visble part of our webpage we will add a Paragraph tag as such:

<!DOCTYPE html> <html>  <head>  </head>  <body>   <p>Text goes here!</p>  </body> </html>

That's all there is to it when it comes to adding simple text on a webpage. Adding the "<p>" with the following closer "</p>" might not be enough. We can get a little bolder and larger with Heading elements.

<!DOCTYPE html> <html>  <head>  </head>  <body>   <h1>A Simple Header</h1>  </body> </html>

Heading elements are scaled 1-6. With "<h1>", "</h1>" being the largest of the set and "<h6>", "</h6>" being the smallest of the set.

Adding Images and Links

All HTML elements can have "Attributes". They give additional information about the elements. They are always put in the starting tag of the element, below is an Anchor element:

<!DOCTYPE html> <html>  <head>  </head>  <body>   <a href="https://www.google.com/"> Click me!</a>  </body> </html>

As we can see there is something different about this element the "href="https://www.google.com/"". This is considered the Attribute of the element. If "Click me!" is clicked it will send the user to the specified webaddress, in this case Google! The only thing visible to the end user is the text within the starting tag, <a>, and the ending tag, </a>, i.e. "Click me!"

Some elements are "Self closing" where they do not need an ending tag. Below we are introduced to an Image Tag:

<!DOCTYPE html> <html>  <head>  </head>  <body>   <img src="https://tinyurl.com/22nyt6yw"/>  </body> </html>

Here we see the attribute is "src="https://tinyurl.com/22nyt6yw"" this url points to a specific image which is then displayed. Note that there is no content or ending tag, the entire element is defined within the starting tag, "<img>".

Ordered and Unordered Lists

Lists are a great way to organize a set of related items and HTML has an element for that! Let's start with a list that is sequential, known as an Ordered List element.

<!DOCTYPE html> <html>  <head>  </head>  <body>   <ol>   </ol>  </body> </html>

To insert an item into the list we'll use a List Item element:

<!DOCTYPE html> <html>  <head>  </head>  <body>   <ol>    <li>First Item<li>    <li>Second Item<li>    <li>Third Item<li>   </ol>  </body> </html>

The List Item element "<li>","</li>" also works for Unordered Lists as well. For Unordered List elements we'll use the starting and ending tag "<ul>", "</ul>" respectively.

<!DOCTYPE html> <html>  <head>  </head>  <body>   <ul>    <li>Item<li>    <li>Item<li>    <li>Item<li>   </ul>  </body> </html>

Ordered List elements and Unordered List elements make up the entirety of lists in HTML.
